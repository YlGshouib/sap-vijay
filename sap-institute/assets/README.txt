ASSETS FOLDER GUIDE
====================

Drop your image files into this folder using the exact names below,
then follow the matching comment in index.html to swap the placeholder
box for a real <img> tag.

logo.png
  - Your institute logo (square, ~120x120px works well)
  - Used in the nav bar and footer (replaces the "S" monogram box)

trainer.jpg
  - Photo of Akshay (portrait orientation, 4:5 ratio looks best)
  - Used in the Trainer section

logos/  (folder)
  - Official logo files for referral partner companies, e.g:
      logos/infosys.png
      logos/pwc.png
      logos/capgemini.png
      logos/accenture.png
      logos/tech-mahindra.png
      logos/tcs.png
  - Swap each <span class="company-badge">Name</span> in index.html for:
      <span class="company-badge"><img src="assets/logos/infosys.png" alt="Infosys" style="height:22px;"></span>
  - Note: use only official/licensed logo assets you have rights to use.

students/  (folder)
  - Student testimonial photos, named to match script.js:
      students/1.jpg
      students/2.jpg
      students/3.jpg
      students/4.jpg
      students/5.jpg
  - Used in the "Student Experiences" 3D sliding carousel
